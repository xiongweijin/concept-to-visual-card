# Prompt Template

The final prompt should be in English. In-image text remains in the user's language.

```text
Create a single {aspect_ratio} visual knowledge card in {style}.

Purpose:
The image explains the concept "{concept_name}" for {audience}, intended for {usage_scene}.

Core idea:
{plain_language_explanation}

Visual direction:
{visual_direction}

Layout:
- A clear title area at the top.
- One central visual metaphor or structured diagram.
- 3 to 5 concise explanation modules around or below the main visual.
- A final takeaway sentence at the bottom.
- Keep the page readable on a phone screen.

In-image text:
Title: "{title}"
Subtitle: "{subtitle}"
Key modules:
1. "{module_1}"
2. "{module_2}"
3. "{module_3}"
4. "{module_4}"
Takeaway: "{takeaway}"

Design requirements:
- All in-image text must be in {language}.
- Use clear, readable Chinese typography if the language is Chinese.
- Avoid clutter and oversized paragraphs.
- Maintain strong visual hierarchy.
- Use consistent icons, arrows, and labels.
- Professional, polished, shareable on social media.
- No random extra text, no watermark, no logo.
```
